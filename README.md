# 🏠 House Price Predictor

This project predicts bangalore house prices using Machine Learning (Ridge Regression).

## 📌 Features
- Data preprocessing
- Model training using Ridge Regression
- Web interface using Flask
- Real-time prediction

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Flask

## 📂 Project Structure
house-price-prediction/
│
├── templates/
│   └── index.html
│
├── model/
│   └── RidgeModel.pkl
│
├── data/
│   └── cleaned_data.csv
│
├── notebook/
│   └── house_price.ipynb

## 🚀 How to Run
1. Clone repo
2. Install requirements
3. Run main.py

## 📊 Model Used
- Ridge Regression
